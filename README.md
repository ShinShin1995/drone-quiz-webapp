# 無人機專業操作證 - 智慧全覆蓋模擬測驗系統

這是一個為無人機專業操作證學科測驗所設計的「智慧全覆蓋模擬測驗系統」單網頁 Web App。

## 🌟 系統特色

1. **單一網頁架構**：所有 HTML、CSS 樣式與 JavaScript 控制邏輯均整合在單一 `index.html` 中，免安裝、免伺服器，直接按兩下即可在瀏覽器中執行。
2. **完整題庫嵌入**：內建民航局官方 114.5.28 最新版本之專業操作證學科測驗題庫，共 **588 題**（含完整題目、A/B/C/D 選項及經校對的正確答案）。
3. **智慧出題機制**：
   - **自訂題數**：可自由選擇一次挑戰 20 題、40 題（標準考試題數）、50 題、100 題或全考。
   - **全覆蓋刷新**：系統會優先隨機抽取「本輪尚未練習過」的題目，確保練習不留死角。當所有題目皆練習過一次後，自動提示並重置練習進度。
   - **錯題高效複測**：答錯的題目會自動加入「錯題本」。在此模式下，系統只會從錯題本中抽題，且「答對後才能將該題從錯題本中移除」。
   - **熟練度評估**：引進連續答對追蹤，題目只要「連續答對 3 次」即會標記為「完全熟練」，並即時呈現在熟練度統計中。
4. **即時儀表板**：顯示題庫總題數、本輪未練習題數、錯題本數量與完全熟練度百分比。
5. **持久化儲存**：所有練習進度、錯題與熟練度資料皆即時透過 `localStorage` 儲存，關閉瀏覽器或重新整理網頁進度亦不會消失。

## 🚀 執行方式

1. 在專案資料夾中，直接雙擊 `index.html` 檔案。
2. 檔案將會在您的預設瀏覽器（例如 Chrome、Edge、Firefox）中開啟。
3. 選擇您要練習的章節與題數，點擊「開始全覆蓋練習」即可開始。

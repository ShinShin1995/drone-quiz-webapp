# 專案開發筆記 - 無人機專業操作證智慧全覆蓋模擬測驗系統

## 📅 開發進度

- **[2026/07/03]** 專案初始化與功能實作：
  - **資料擷取**：完成對民航局 114.5.28 最新版「專業操作證學科題庫」的資料擷取。直接從官方 `.docx` 格式進行解析，確保 588 題中所有中文字元、選項結構、各章節分類與答案對應達到 100% 精準度。
  - **系統機制**：
    - 實作自訂題數（20/40/50/100/All）與章節篩選。
    - 實作本輪未練習題優先抽取（全覆蓋）與自動重置機制。
    - 實作錯題本（答錯加入，錯題模式下答對移除）。
    - 實作連續答對 3 次標記「完全熟練」之熟練度評估機制。
    - 整合 `localStorage` 進行練習資料持久化。
    - 實作即時更新的頂部數據儀表板。
  - **UI/UX 設計**：使用 Outfit 與 Noto Sans TC 字型，搭配深色科技 Slate 質感背景、玻璃擬態（Glassmorphism）卡片、醒目的答題高亮反饋與圓角邊框，打造現代美觀的響應式前端網頁。

## ⚙️ 系統資料結構

### 題庫資料格式 (database)
```javascript
{
    id: 題號(數字),
    chapter: 章節編號(1-4),
    chapter_name: "章節名稱",
    question: "題目完整敘述",
    options: ["(A) ...", "(B) ...", "(C) ...", "(D) ..."],
    answer: "正確答案(A/B/C/D)"
}
```

### LocalStorage 狀態格式 (`drone_quiz_advanced`)
```json
{
    "practicedIds": [1, 2, 5],
    "wrongIds": [3, 4],
    "mastery": {
        "1": 3,
        "2": 1,
        "5": 0
    }
}
```
*(其中 `mastery` 中的值代表該題連續答對次數，若答錯會歸零。)*
